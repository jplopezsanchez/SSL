# 00 - Hello World en C

## Programa simple que imprime "Hola Mundo".

### Compilador utilizado

gcc (GNU Compiler Collection)

### Versión de C

El programa fue compilado utilizando el estándar C23 mediante la opción `-std=c23` del compilador gcc:
`gcc -std=c23 hello.c -o hello`

#### Luego se ejecutó el programa con:
`./hello`

#### Obteniendo como salida:
hola mundo


### Versión del compilador
gcc.exe (Rev13, Built by MSYS2 project) 15.2.0

## Obtención de la versión del compilador

Se utilizó el comando:

`gcc --version`
